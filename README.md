# mehmetkucuk.nl

Freelance Yazılım ve Web Tasarım Uygulaması.

## Özellikler

- **%100 SEO Uyumlu**: Next.js App Router ve SSR ile optimize edilmiştir.
- **Çoklu Dil Desteği**: Türkçe (varsayılan), İngilizce ve Hollandaca desteği (altyapı hazır).
- **Google Mantığı Tasarım**: Basit, hızlı ve kullanıcı dostu arayüz.
- **Teknolojiler**: Next.js 15, Tailwind CSS, TypeScript, next-intl.

## Kurulum

Geliştirme sunucusunu başlatmak için:

```bash
npm run dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresine gidin.

## Yapı

- `src/app/[locale]`: Sayfa ve layout dosyaları (i18n routing).
- `messages/`: Dil dosyaları (tr.json, en.json, nl.json).
- `src/components`: UI bileşenleri.
